Math IA — Statistical Analysis of Azul Bot Strategies

IB DP Mathematics Internal Assessment — Exploring game theory through simulation

Research Question

How do different heuristic strategies affect win rates, score distributions, and game length in the board game Azul?

This IA uses a simulation platform that plays thousands of Azul games between AI bots using different strategies. All game data is stored in a database and can be analysed using SQL queries, Python scripts, or the built-in visualisation tools.
